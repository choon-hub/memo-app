# レスポンシブ対応（PC 幅での間延び解消）

以下の内容を実装してください。

## 背景

`@media` クエリがプロジェクト全体に1つもなく、`.page`（`app/assets/global.css`）や各カードは固定 px 前提のモバイルレイアウトになっている。タブレット・PC 幅で表示すると `max-width` 制約がないため間延びする。

## 変更内容

1. `app/layouts/default.vue` のレイアウトコンテナに `max-width: 480px; margin: 0 auto;` 程度の制約を加え、PC ブラウザでも中央寄せのモバイル幅で表示されるようにする
2. 全ページ（`/one-new` `/topics` `/workout`）で崩れがないか確認し、必要ならカード・フォームの幅指定を追従修正する

## 対象範囲・制約

- 対象：`app/layouts/default.vue`、必要に応じて `app/assets/global.css`
- 本格的なブレークポイント設計（タブレット/PC 向けの複数カラム化など）はスコープ外。中央寄せ＋最大幅の最小対応に留める

## 完了条件

- `npm run test` と `npm run lint` が通る
- `npm run dev` で起動し、PC 幅（1280px 程度）で3ページを表示してレイアウトが中央寄せになっていることをスクリーンショットで確認する（`/run` スキルが利用可能）
