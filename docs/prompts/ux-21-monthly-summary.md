# 月次・週次の振り返りサマリー

以下の内容を実装してください。

## 背景

記録が単なるログで終わらず振り返りに使えるよう、「1日1新」「トピック」の記録数推移や「筋トレ」の部位（カテゴリ）別トレーニング頻度を月次でまとめて表示する。`ux-08-streak-heatmap.md`（ヒートマップ）実装後の自然な拡張として位置づけられている。

## 前提

`ux-08-streak-heatmap.md` が実装済みであることを前提とする。未実装の場合はその旨を報告し、先に ux-08 を実施するか確認を取る。

## 変更内容

1. 月次サマリーを表示するセクション（またはページ）を追加する
   - 1日1新・トピック：当月の記録数と前月比
   - 筋トレ：当月のカテゴリ（chest/back/legs）別の記録回数
2. 集計は既存の取得データから行う純粋関数として `app/utils/` に実装する
3. 表示位置は各ページ内のヒートマップ近くとする。独立ページ（例：`/summary`）にした方が見通しが良いと判断した場合は、理由を添えてユーザーに確認してから進める

## 対象範囲・制約

- グラフ描画が必要な場合は依存ライブラリを追加せず SVG で行い、実装前に `dataviz` スキルを読む
- 週次表示は月次が完成してから検討する（本プロンプトでは月次のみ）
- 目標設定・達成率などの機能はスコープ外

## 完了条件

- 集計関数の単体テスト（月境界・データなし月・カテゴリ別集計）を追加する
- `npm run test` と `npm run lint` が通る
